Detailed Description

template<typename T, typename Compare = std::less<T>>
struct modelbox::StableElement< T, Compare >

QueueType: FifoQueue, PriorityQueue, StablePriorityQueue. FifoQueue: fisrt in, first out. PriorityQueue: priority queue. StablePriorityQueue: stable ordered queue, when priority same, first in, first out.

StableElement: to keep elements in order which have same priority.
