Detailed Description

template<typename T, typename Compare = std::less<T>, typename Sequence = std::vector<T>>
class modelbox::PriorityBlockingQueue< T, Compare, Sequence >

Priority Blocking queue, blocking queue with stable priority. Support geting all elements with same priority in one batch.
