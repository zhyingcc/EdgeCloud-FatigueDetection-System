335 /*
336  Log may crash when the shared library loaded by deepbind and call this log
337  API. The reason is symbol copy-relocation, About copy-relocation, read here:
338  https://stackoverflow.com/questions/37296995/dynamic-loading-of-shared-library-with-rtld-deepbind
339  Setting the symbol to weak allows the deepbind library to use the first symbol
340  without error. If there is still a problem, you can use macros
341  MBLOG_LIBRARY_DEEPBIND. but this will cause a little slower when calling log
342  API.
343 */
344 #ifdef MBLOG_LIBRARY_DEEPBIND
345 #define ModelBoxLogger modelbox::GetLogger()
346 #else
347 #define ModelBoxLogger modelbox::klogger
348 #endif
